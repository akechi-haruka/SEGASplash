Configurable splash screen that plays a SEGA video.
Can be used to cover things starting up in the background.

Screenshot: https://puu.sh/K4IGf/51d4f968df.png

Command line args:

  --volume            (Default: 50) Changes video volume
  --min-wait          (Default: 3) Minimum seconds to wait
  --windowed          (Default: false) Run windowed
  --skip-on-window    (Default: false) If the wait-for window is found, skip the
                      min-wait timer
  --wait-for          Show video until a window with this title is detected
  --help              Display this help screen.
  --version           Display version information.
